Минимальная сумма для начала работы с платформой складывается из стоимости приобретенного пакета (на выбор их три) и из минимально выделенного депозита на вашем биржевом аккаунте роботу для торговли (на одного фьючерсного робота минимум 200 usdt). Для получения наилучших результатов мы рекомендуем депозит в вашем аккаунте на бирже не менее 3000 usdt для диверсификации торговых пар (чем больше тем лучше). Минимальная сумма для копитрейдинга 500 usdt.
